## Lists

Unordered lists:

- Item 1
- Item 2

Ordered Lists:

1. Item 1
1. Item 2
1. Item 3
1. Item 4
1. Item 5

Task Lists:

- [ ] task
  - [x] sub task
  - [ ] sub task
- [x] task

Some things to note: task lists can be interacted with on confluence, but they
will be overwritten when the markdown is synced again.
